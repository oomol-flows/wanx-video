# 万相生视频

使用阿里万相 AI 模型生成视频，支持 4 种模式：文生视频、图生视频、关键帧生视频、参考生视频。

## 功能特性

### 文生视频
使用万相 AI 从文本描述生成视频。支持多种分辨率（720P 至 1080P）、时长（2-15 秒）和镜头类型（单镜头/多镜头）。

### 图生视频
从首帧图像生成视频。可选择 flash 模型（速度快）或 standard 模型（质量高）。支持自定义音频和多种分辨率。

### 关键帧生视频
从首尾帧图像生成过渡视频。非常适合在两张图像之间创建平滑过渡，支持特效模板。

### 参考生视频
从参考素材（图像或视频）生成视频。在提示词中使用角色占位符（character1、character2）引用上传的素材。

## 模块

| 模块 | 描述 |
|------|------|
| `text2video` | 从文本提示词生成视频 |
| `image2video` | 从首帧图像生成视频 |
| `keyframe2video` | 从首尾帧生成过渡视频 |
| `reference2video` | 从参考素材生成视频 |
